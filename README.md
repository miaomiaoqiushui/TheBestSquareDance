# 最炫广场舞简介

## 项目简述
广场舞在国内越来越受到广大中老年群体的喜爱，舞群组织不规范导致扰民问题时有发生、舞蹈教学视频缺乏导致广场舞爱好者训练难等问题。为了解决上述问题，设计和研发了最炫广场舞APP。

## 功能概述

### Android用户端
实现功能分为四大模块：舞群导航、视频聚合、舞具商城和个人中心。

### Web管理端
实现功能分为：管理员登陆、注册以及用户后台数据的增删查改操作。
